## 080. 영어 끝말잇기

## 풀이

룰에 맞춰 문자열이 이어지지 않거나, 이전에 사용된 단어가 있는 경우 사람수를 고려하여 답을 리턴한다.

## 생각

prev라는 변수를 두어 이전 단어를 저장하여 문자열이 이어지는지를 판단하고, 인덱스 기반으로 현재 단어가 이전에 등장했는지를 판별한다.

만약 룰을 위반하였을 경우 현재 단어의 순번을 사람수로 나누어 몫과 나머지로 누가 몇번째로 틀렸는지를 구할 수 있다.

## 다른 사람 풀이
이건 좀...
```
def solution(n, words):
    for p in range(1, len(words)):
        if words[p][0] != words[p-1][-1] or words[p] in words[:p]: return [(p%n)+1, (p//n)+1]
    else:
        return [0,0]
```
